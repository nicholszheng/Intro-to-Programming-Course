Adding style to this page

To add style to this page, first make sure you place < style > < /style > in the head section. Here are some of the following styles I applied to this page:

  • bolded text with [font-weight: bold;]
  • wanted to make a title specifically font sized in 30pt [font-size: 30pt;]
  • changed text color to green with [color: green;]
  • underlined text with [text-decoration: underline;]
  • created a class to center whatever I choose [.center] with [text-align: center;] (separate style element)
  • centered my main title and Nest List exercise, one example: < h1 class = "center" > My Notes by Nicole Zheng < / h1 > (inline style attribute)
  • added a margin around the whole page by making body as a selector in the style with [margin:50px;]
  • made a title colored blue in the same line as an already existing attribute for example [< h3 class = "center" ; style = "color : blue ; " > Nest List exercise < / h3 >] (inline style attribute)
  • use the < span > < / span > element to specifically color a word in a sentence blue [ < span style = "color: blue"> WORD < /span >]
